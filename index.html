nikic's PEAR channel

Registering the channel: 
pear channel-discover nikic.github.com/pear
Listing available packages: 
pear remote-list -c nikic
Installing a package: 
pear install nikic/package_name
Installing a specific version/stability: 
pear install nikic/package_name-1.0.0
pear install nikic/package_name-beta
Receiving updates via a feed: 
http://nikic.github.com/pear/feed.xml

The nikic.github.com/pear PEAR Channel Server is proudly powered by Pirum 1.1.4
